Cómo crear un diagrama de dispersión

  • Introduzca el título del gráfico.
  • Para cada serie, ingrese valores de datos con delimitador de espacio, etiqueta, color y tipo de línea de tendencia.
  • Para cada eje, ingrese el valor mínimo del eje, el valor máximo del eje y la etiqueta del eje.
  • Presione el botón Dibujar para generar el diagrama de dispersión.
  • Pulse el botón de reinicio × para establecer los valores predeterminados.
