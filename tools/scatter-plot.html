How to create a scatter plot

  • Enter the title of the graph.
  • For each series, enter data values with space delimiter, label, color and trendline type.
  • For each axis, enter minimal axis value, maximal axis value and axis label.
  • Press the Draw button to generate the scatter plot.
  • Press the reset button to set default values.
